# Student Task Tracker

Simple Full Stack DevOps Project

## Technologies Used

- React
- Node.js
- Express
- Docker
- Jenkins
- Azure
- Vercel

---

## Features

- Task Display
- REST API
- Docker Containerization
- CI/CD Pipeline
- Security Check using npm audit

---

## DevOps Workflow

1. Developer pushes code to GitHub
2. Jenkins pulls repository
3. Jenkins installs dependencies
4. Jenkins performs security check
5. Docker image is built
6. Application is deployed
7. Frontend hosted on Vercel
8. Backend hosted on Azure
